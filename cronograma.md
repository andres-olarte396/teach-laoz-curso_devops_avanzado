# CRONOGRAMA DETALLADO: DEVOPS ENGINEER (ADVANCED)

## RESUMEN TEMPORAL
- **Duración Total**: 48 Semanas (12 Meses)
- **Carga Semanal**: 8-10 Horas
- **Fecha de Inicio Sugerida**: Flexible (Semana 1)

## CALENDARIO SEMANAL

### MES 1: INTRODUCCIÓN A DEVOPS
- **Semana 1**: Fundamentos del Movimiento (Historia, Wall of Confusion).
- **Semana 2**: Principios CALMS y Lean (Teoría aplicada).
- **Semana 3**: Roles, Responsabilidades y Anti-patrones.
- **Semana 4**: **Entregable**: Análisis crítico de un pipeline real.

### MES 2: CONTROL DE VERSIONES AVANZADO (GIT)
- **Semana 5**: Git Internals y Estrategias de Ramificación (GitFlow vs Trunk-Based).
- **Semana 6**: Monorepos vs Polyrepos y Gestión de Dependencias.
- **Semana 7**: Automatización con Git Hooks y Políticas de PR.
- **Semana 8**: **Entregable**: Migración de repositorio a Trunk-Based.

### MES 3: CI/CD FUNDAMENTOS
- **Semana 9**: Integración Continua (Pipelines Declarativos, Linters).
- **Semana 10**: Entrega Continua (Artefactos, Versionado SemVer).
- **Semana 11**: Herramientas de CI/CD (GitHub Actions/GitLab CI).
- **Semana 12**: **Entregable**: Pipeline CI con pruebas y análisis estático.

### MES 4: INFRAESTRUCTURA COMO CÓDIGO (IaC)
- **Semana 13**: Fundamentos IaC (Terraform/Pulumi, Declarativo vs Imperativo).
- **Semana 14**: Gestión del Estado (State) y Backends Remotos.
- **Semana 15**: Modularidad, DRY y Gestión de Entornos (Workspaces).
- **Semana 16**: **Entregable**: Infraestructura replicable en Staging y Prod.

### MES 5: CONFIGURACIÓN Y AUTOMATIZACIÓN
- **Semana 17**: Gestión de Configuración (Ansible/Chef) e Inventarios.
- **Semana 18**: Idempotencia y Playbooks Avanzados.
- **Semana 19**: Detección de Drift y Remediación Automática.
- **Semana 20**: **Entregable**: Automatización completa del provisionamiento.

### MES 6: CONTENEDORES (DOCKER)
- **Semana 21**: Docker Internals (Namespaces, Cgroups, UnionFS).
- **Semana 22**: Best Practices (Multi-stage Builds, Distroless).
- **Semana 23**: Seguridad en Contenedores y Gestión de Secretos.
- **Semana 24**: **Entregable**: Imagen optimizada y segura.

### MES 7: ORQUESTACIÓN (KUBERNETES)
- **Semana 25**: Arquitectura K8s (Control Plane, Workers, Etcd).
- **Semana 26**: Objetos Básicos (Pod, Service, Ingress, ConfigMap).
- **Semana 27**: Operación Avanzada (Helm Charts, Autoscaling HPA/VPA).
- **Semana 28**: **Entregable**: App desplegada con Helm y Autoscaling.

### MES 8: OBSERVABILIDAD (O11Y)
- **Semana 29**: Los 3 Pilares: Métricas (Prometheus) y Logs (ELK/Loki).
- **Semana 30**: Tracing Distribuido (OpenTelemetry) y Context Propagation.
- **Semana 31**: Visualización (Grafana) y Diseño de Dashboards.
- **Semana 32**: **Entregable**: Dashboard operativo + alertas eficientes.

### MES 9: SEGURIDAD (DEVSECOPS)
- **Semana 33**: Shift-Left Security (SAST, DAST, SCA en CI).
- **Semana 34**: Seguridad en la Cadena de Suministro (SBOM, Signing).
- **Semana 35**: Seguridad en Runtime (Falco, Network Policies).
- **Semana 36**: **Entregable**: Pipeline CI/CD con controles de seguridad.

### MES 10: CLOUD (AWS/AZURE/GCP)
- **Semana 37**: Fundamentos Cloud (IAM, Identidad, Least Privilege).
- **Semana 38**: Networking Cloud (VPC, Subnets, Peering).
- **Semana 39**: Servicios PaaS y FinOps (Control de Costos).
- **Semana 40**: **Entregable**: Infraestructura en cloud productivo.

### MES 11: SRE Y CONFIABILIDAD
- **Semana 41**: Midiendo Confiabilidad (SLIs, SLOs, SLAs).
- **Semana 42**: Error Budgets y Gestión de Riesgo.
- **Semana 43**: Gestión de Incidentes y Postmortems Blameless.
- **Semana 44**: **Entregable**: Diseño de SLOs + Simulación de incidente.

### MES 12: PROYECTO FINAL INTEGRADO
- **Semana 45**: Planificación y Arquitectura del Proyecto Final.
- **Semana 46**: Implementación (Construcción de Pipeline e Infraestructura).
- **Semana 47**: Operación Simulada (Chaos Engineering e Incidentes).
- **Semana 48**: **Defensa Final**: Evaluación de calidad, seguridad y resiliencia.
